# DeepScholar AI

DeepScholar AI is an Android application that combines AI-assisted research, note-taking, project management, and study planning into a single workspace. The goal of this project is to provide a clean and organized environment where users can research topics, manage knowledge, and stay productive without switching between multiple apps.

The project is built using **Kotlin**, **Jetpack Compose**, **Material 3**, and follows the **MVVM** architecture with a modular service layer to make future AI integrations easier.


## Features

### Research Assistant

* Generate detailed research on any topic
* Structured responses with sections like Overview, Analysis, Key Insights, and Conclusion
* Summarize concepts
* Compare topics
* Generate study notes
* Suggest related topics
* Generate follow-up research questions


### My AI Assistant

A separate chatbot module designed for everyday conversations and productivity.

Features include:

* Chat history
* Search chats
* Pinned conversations
* Built-in prompt templates
* Independent service layer for future AI integrations

Prompt templates include:

* Explain Like I'm 5
* Summarize Text
* Generate Study Notes
* Generate Quiz Questions
* Brainstorm Ideas
* Simplify Technical Concepts


### PDF & Audio Support

The chatbot can work with uploaded files.

Supported features:

* Read PDFs
* Summarize research papers
* Extract key insights
* Answer questions about documents
* Generate notes
* Summarize audio recordings

Uploaded PDFs can also be opened using the phone's default PDF viewer.


### Research Projects

Create projects to organize long-term research.

Each project stores:

* AI responses
* Notes
* Research history
* PDFs
* Bookmarks


### Notes

The notes section is organized using a workspace structure.

* Workspace
* Folders
* Notes
* Pdfs
  
Features:

* Rich text editor
* Code blocks
* Checklists
* Tables
* Tags
* Search
* Auto-save
* Favorites

Notes open in a full-screen editor for easier editing.


### Research Vault

A simple library for storing research papers.

Users can:

* Organize papers into folders
* Bookmark papers
* Add notes
* Open PDFs


### Planner

The planner helps organize study sessions and research work.

Features:

* Daily tasks
* Weekly goals
* Monthly goals
* Deadlines
* Calendar view
* Progress tracking

Assistant Mode can estimate how long a topic may take to learn and generate a study schedule. The schedule can be added to the planner after user approval.


### Dashboard

The dashboard provides a quick overview of:

* Recent projects
* Notes
* Saved papers
* Planner
* Productivity statistics
* AI activity


### Search

Global search is available across:

* Projects
* Notes
* Research papers
* Planner
* Chat history


### Authentication

* Login
* User registration
* Username and password authentication


## UI

The application uses a clean Material 3 interface.

* Dark Mode by default
* Optional Light Mode
* Smooth navigation
* Simple animations
* Research-focused layout


## Tech Stack

* Kotlin
* Jetpack Compose
* Material 3
* MVVM Architecture
* Repository Pattern
* Navigation Component
* Local Storage
